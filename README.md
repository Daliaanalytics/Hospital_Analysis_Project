# Hospital_Analysis_Project
## Overview
Analysis of hospital patient data to identify trends and insights 
to help improve healthcare services.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Power BI

## Key Findings
- Flu is the most common disease 
- Heart disease has the longest average stay(8.2 day)
- Heart disease is the most expensive treatment (6300 Avg)
- 75% of patients recovered sucessfully
- General department is the busiest (30%)
- Older patients (+51) are more prone to heart disease
- Heart disease is more common in males, diabetes in females.
